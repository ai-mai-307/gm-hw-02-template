# Домашнее задание №3. Обучение с подкреплением. Deep Q-Learning

## Описание задания

На лекции по обучению с подкреплением мы познакомились с идеей Q-learning и переходом от табличного представления Q-функции к нейросетевой аппроксимации. В этом задании ваша задача заключается в том, чтобы реализовать алгоритм **Deep Q-Learning (DQN)** и обучить агента играть в одну из консольных игр Atari.

## Постановка задачи

В рамках задания вам необходимо выбрать одну среду Atari из предложенного списка, реализовать DQN-агента, обучить его и продемонстрировать результат.

### 1. Подготовка среды: 1 балл

1. Установите необходимые зависимости для работы с Atari-средами через `Gymnasium` и `ALE`.
2. Зарегистрируйте Atari-среды из `ale_py`.
3. Создайте **одну** среду из списка рекомендованных:
   * `ALE/Pong-v5`

   * `ALE/Freeway-v5`

#### Требования к среде, если вы хотите выбрать её самостоятельно

Помимо рекомендованного списка, допускается использование другой игры Atari из семейства ALE, доступной через `Gymnasium` по идентификатору вида `ALE/<Game>-v5`.

При самостоятельном выборе среды необходимо соблюдать следующие требования:

1. Среда должна быть **однопользовательской Atari-средой ALE**, совместимой с API `Gymnasium`. Не допускается использование multi-agent сред.
2. Пространство действий должно быть **дискретным и конечным**. Не допускается использование непрерывного action space (`continuous=True`).
3. Необходимо использовать **стандартный набор действий среды**, предоставляемый по умолчанию. Использование `full_action_space=True` не допускается.
4. Не допускается использование нестандартных настроек среды, если они не были отдельно согласованы с преподавателем. В частности, не следует менять параметры `mode`, `difficulty`, `continuous`, `continuous_action_threshold`.
5. В качестве наблюдений рекомендуется использовать визуальное представление игры: `obs_type="rgb"` или `obs_type="grayscale"`.
6. При выборе собственной среды в блокноте необходимо кратко обосновать выбор игры и явно указать:
   * идентификатор среды;
   * размерность `observation_space`;
   * тип и мощность `action_space`;
   * используемые параметры создания среды.

### 3. Реализация DQN-агента: 3 балла

Реализуйте алгоритм **Deep Q-Learning** с использованием нейросети в качестве аппроксимации Q-функции.

Ваша реализация должна содержать следующие компоненты:

1. **Нейросетевая модель** `Q(s, a, θ)`:
   * на вход получает состояние среды;
   * на выходе возвращает оценки Q-функции для всех допустимых действий;
   * для изображений Atari рекомендуется использовать сверточную нейронную сеть.

2. **Replay Buffer**:
   * хранение переходов вида `(s, a, r, s', done)`;
   * возможность случайной выборки мини-батча переходов.

3. **Epsilon-greedy стратегия**:
   * с вероятностью `ε` агент выбирает случайное действие;
   * с вероятностью `1 - ε` — действие с максимальным значением `Q(s, a, θ)`.

4. **Target Network**:
   * отдельная копия основной Q-сети;
   * используется только для вычисления Bellman target;
   * обновляется периодически копированием весов основной сети.

5. **Bellman target и функция потерь**:
   * для одного перехода таргет должен иметь вид  
     `y = r + γ max_a' Q_target(s', a')`,
     а для терминального состояния — только `y = r`;
   * в качестве функции потерь используйте `MSELoss` или `SmoothL1Loss`.

### 4. Обучение агента: 2 балла

Реализуйте цикл обучения, который:

1. взаимодействует со средой;
2. сохраняет переходы в replay buffer;
3. после накопления достаточного числа переходов начинает обучать сеть на случайных мини-батчах;
4. периодически обновляет target network;
5. сохраняет историю обучения:
   * reward по эпизодам;
   * средний reward на окне из нескольких эпизодов;
   * значение loss.

Для каждого эксперимента необходимо:
